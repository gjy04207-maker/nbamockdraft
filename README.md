# NBA 评论生成网站

这是一个分层架构的球评生成系统：
- 风格引擎：学习你的历史球评文风
- 数据检索：从多站点获取事实数据
- 规则校验：基于 CBA 条款进行约束与引用
- 事实核查：对生成内容进行出处追踪
- 写作工作台：左侧对话、右侧预览、可引用再提问

## 目录结构
- `apps/web`: 前端（Next.js）
- `apps/api`: 后端（FastAPI）
- `data/cba/raw`: 原始 CBA 文件
- `data/cba/parsed`: 解析后的 CBA 知识库
- `scripts`: 数据处理与抓取脚本
- `docs`: 方案与规格说明
