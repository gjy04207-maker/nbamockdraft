# 写作工作台 UI 规格

## 总览
- 页面进入后先询问："今天想写点什么？"
- 用户自然语言回答后，系统开始检索资料并构建草稿
- 左侧对话区：你与 AI 的对话、质疑与纠错
- 右侧预览区：实时更新文章
- 支持在预览区选中某段文本并发起追问

## 交互流程
1. 启动页提问
2. 用户输入意图
3. 系统开始“资料检索 + 错误检查 + 逻辑补全”
4. 右侧生成预览
5. 用户选择段落并追问
6. 系统局部重写并保留引用

## 右侧预览区能力
- 段落级引用标记
- 点击引用展开来源
- 支持段落锁定（不被后续生成覆盖）

## 错误提示机制
- 若用户描述与数据冲突，系统必须高亮提示
- 提示包含：冲突字段 + 数据来源
