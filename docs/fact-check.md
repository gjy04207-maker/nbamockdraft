# 事实校验与引用溯源模块

## 目标
- 在生成内容中发现“来源缺失/冲突”的事实
- 给出可追溯的引用提示

## API
- `POST /facts/check`

请求示例：
```json
{
  "draft": "湖人最近5场防守效率下降到108。",
  "sources": [
    {
      "source": "nba.com",
      "facts": ["湖人最近5场防守效率为108.4", "对手三分命中率38.1%"]
    }
  ]
}
```

响应示例：
```json
{
  "issues": [
    {"type": "missing_support", "message": "数字 108 未在提供的来源中出现。", "span": "108"}
  ]
}
```

## 规则（当前）
- 只做轻量级启发式：识别数字 token 是否在来源中出现
- 结果仅供提示，不作为最终结论
